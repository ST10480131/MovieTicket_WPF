Instructions on how to Run the Application
Open Visual Studio
Click Create New Project
Select WPF App
Name it MovieTicket_WPF
Replace the default code with the provided files
Press F5 to run the application

 Validation Rules
 
Customer name must not be empty
Number of tickets must be a positive number
 Example Output
Customer Name: John
Movie: Avatar
Ticket Type: VIP
Quantity: 2
Price per Ticket: R140
Total Amount: R280

 Learning Outcomes

By completing this project, you will understand:

How to design WPF interfaces using XAML
How to handle button click events
How to retrieve data from UI controls
Basic input validation techniques
